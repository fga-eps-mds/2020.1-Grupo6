## Instalação

> * ``` git clone https://github.com/fga-eps-mds/2020.1-Grupo6.git``` <br> <br>

### Guia de Instalção do MkDocs

## Execução

### Para executar um servidor local para testes

> ``` mkdocs serve ```

### Para subir as alterações da documentação no Github Pages

> ``` mkdocs gh-deploy ```
